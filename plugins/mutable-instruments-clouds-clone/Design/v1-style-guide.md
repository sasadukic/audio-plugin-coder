# Style Guide v1

## Design Source
- Library Design: NeoGrid Minimal
- Library ID: neogrid-minimal
- Vibe: Geometric, grid-based, futuristic minimalism

## Color Tokens
- --bg: #111115
- --surface: #1E1E24
- --surface-hi: #2A2A33
- --primary: #00FFFF
- --secondary: #FF0099
- --text-primary: #F0F0F5
- --text-secondary: #A0A0B0
- --border: #333340

## Typography
- Family: Inter, Segoe UI, sans-serif
- Title: 18px / 700 / letter-spacing 0.08em
- Labels: 11px / 600 / letter-spacing 0.1em
- Values: 13px / 600

## Spacing and Geometry
- Outer padding: 20px
- Panel radius: 6px
- Control card radius: 4px
- Grid gap: 16px
- Border width: 1px solid var(--border)

## Control Visual Rules
- Rotary controls use circular track + neon value arc.
- Active control highlights with cyan glow.
- Value readouts are monospace to stabilize width while changing values.

## UI States
- Idle: low glow, neutral text.
- Hover: subtle cyan outline.
- Active/Drag: stronger glow and brighter value text.
