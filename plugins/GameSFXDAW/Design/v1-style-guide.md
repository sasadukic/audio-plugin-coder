# Style Guide v1

## Visual Direction
Theme: `Technical Studio Console`

The interface should feel like a production tool: dense information hierarchy, strong contrast, and clear state accents for draft/in-review/approved assets.

## Typography
- Primary: `Space Grotesk`
- Secondary: `IBM Plex Sans`
- Monospace: `IBM Plex Mono`

## Palette
- Background: `#0d1018`
- Surface: `#161b27`
- Surface Alt: `#1f2636`
- Border: `#2e3950`
- Text Primary: `#ecf1ff`
- Text Secondary: `#9db0d4`
- Accent: `#59d4a7`
- Warning: `#ffb454`
- Danger: `#ff6f7d`

## State Colors
- Draft: `#7a8cb3`
- In Review: `#ffb454`
- Approved: `#59d4a7`

## Motion
- Use short transitions (120-180ms) for hover/focus/selection.
- Use subtle elevation and border glow for active container card.
